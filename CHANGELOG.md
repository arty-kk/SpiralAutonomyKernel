# Changelog

## 1.0.0 - 2026-03-30

Initial public release of Spiral Autonomy Kernel.

Highlights:

- repositioned the project around bounded autonomous self-evolution,
- removed vague and misleading public framing,
- translated prompts and runtime-facing text to English,
- added Apache 2.0 licensing and release collateral,
- documented architecture, guarantees, deployment, and evaluation,
- added reproducible evidence generation,
- added focused smoke and policy tests for the release surface,
- retained the `sif` CLI alias for compatibility.
