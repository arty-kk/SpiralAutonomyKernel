# Roadmap

## Near term

- strengthen process-level observability around autonomous decisions,
- expand replayable evidence for multi-cycle runs,
- improve prompt contracts for plan and reflection tasks,
- add policy profiles for stricter production deployments.

## Medium term

- add richer evaluation adapters for domain-specific signals,
- support explicit operator approval gates for selected changes,
- expand structured reporting for long unattended sessions,
- improve isolation around generated code surfaces.

## Longer term

- add benchmark scenarios for multi-cycle convergence,
- expose a minimal service interface for remote orchestration,
- support more granular self-modification scopes and policy profiles.
